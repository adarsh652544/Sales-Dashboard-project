# Sales Performance Dashboard

## 📊 Overview
This project presents an interactive Sales Dashboard built using Excel and Power BI.

## 🛠 Tools Used
- Microsoft Excel
- Power BI

## 📌 Key Insights
- West region generated the highest sales
- Technology category performed best
- Sales peaked during year-end months
- Some products showed negative profit

## 📁 Files
- Power BI Dashboard (.pbix)
- Dataset (.xlsx)
- Dashboard Screenshot

## 🚀 Objective
To analyze sales data and create a dashboard for better business decision-making.
